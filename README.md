# AlternatePerspective ([Nexus Mods Mirror](https://www.nexusmods.com/skyrimspecialedition/mods/50307))

An overhaul to Skyrims Intro Sequence, artificially delaying the execution scene to allow a variety of alternate starting scenarios and the option to view Alduin's appearance from the perspective of a regular civilian instead of the well known Prisoner PoV.

Alternate Starting scenarios use a custom made user interface, the source code for which can be viewed [here](https://github.com/KrisV-777/AlternatePerspective-MessengerMenu).  
The alternate starting options are expandable by using `.json` config files. Further information on this can be found in the [wiki](https://github.com/KrisV-777/Alternate-Perspective/wiki).

# Core Features
* Delayed intro sequence, allowing script initialization and a faster start into the game
* Custom, alternate intro sequence to view the execution scene from an outsider's perspective
* Expandable alternate start system using a custom user interface to offer a variety of ways to start the game beyond the default Inn start
  * The Vanilla Intro Quest and various Main Quest skips can be found as custom starting scenarios here
* A fully explorable Helgen showcasing what Helgen could have been prior to Alduin's attack
